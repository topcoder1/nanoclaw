Top 5-7 competitors in the self-hosted personal AI agent space, with detailed architectural comparison.

- AgentGPT / Reworkd
- LobeChat

- crewAI
- Eliza / ElizaOS (ai16z)
- Home Assistant AI integrations
- Continue.dev (coding focus)
- anything-llm
- Local AI / LocalAI
- Jan.ai
- Msty

## Key dimensions
1. GitHub stars / adoption
2. Core value prop
3. Architecture (how agents run, isolation model, LLM support)
4. Channel support (messaging, web UI, API)
5. Strengths/weaknesses
6. Feature matrix: container isolation, multi-channel, browser automation, memory/learning, trust framework, parallel execution
